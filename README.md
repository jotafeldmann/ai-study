# AI Study

A small repository for studying applied AI concepts and building AI Developer skills using Javascript.

## About
Practical experiments, sample projects, and notes focused on applying AI techniques (ML, embeddings, prompt engineering, model integration) in JavaScript environments.

## Tech
- JavaScript / Node.js
- npm
- Common AI libraries and APIs (listed per project)

## Getting started
1. Clone the repo:
    git clone <repo-url>
2. Install dependencies:
    npm install
3. Run examples:
    npm run start  # or see each example's README

## Repo structure
- /module-X: each module is a different part or skill
